web technologies

HTML

HTML (HypterText Markup Language) is the fundamental building block of the Web and defines the content and structure of a web page. Below are some common page elements that HTML is used for:

  • Headings
  • Paragraphs
  • Images
  • links
  • lists

Learn more about HTML with the w3Schools HTML Tutorial

CSS

CSS (Cascading Style Sheets) is used todescribe a web page's appearance and presentation. Below are some commonuses of CSS:

  • Fonts
  • colors
  • Layout
  • Backgrounds

Learn more about CSS with the w3schools CSS Tutorial

JAVASCRIPT

JAVASCRIPT JavaScript is the programming language of the web and controls the functionality and behaviour of a web page. Below are some common uses of JavaScript:

  • add interactivity to a web page
  • create web apps
  • create web games and animations

A great JavaScript reference is The Modern JavaScript Tutorial.
